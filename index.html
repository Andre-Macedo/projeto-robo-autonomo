Sobre o Projeto

Carro autônomo com Arduino e sensores ultrassônicos construido para competir na Robocar Race 2024.

Componentes

Sensor Ultrasonico

Sensores ultrasonicos para controlar o carro de forma precisa.

Navegação Autonoma

Navegação autonoma com sensores ultrasonicos e servo motor.

Arduino

Arduino integra os sensores com os motores.
